# Advanced Skybox

Advanced Skybox displays user-supplied local video frames as a configurable RuneLite overlay while playing Old School RuneScape.

It is designed for visual skybox ambience, trippy animated backgrounds, picture-in-picture style viewing, and captioned shows or movies where subtitles are already burned into the frames.

**Video-only:** This Plugin Hub version does not play audio. If the media requires dialogue or audio context, create captioned/subtitled frames before using them with the plugin.

**Media Disclaimer:** This plugin does not download, stream, host, or provide any media. Users are responsible for creating and supplying their own local media files.

## Features

- Displays extracted image frames as an overlay
- Visual playback controls for local frame packs
- Timeline slider and 10-second visual seek controls
- Adjustable size, position, opacity, alignment, and bottom fade
- Supports full-screen or custom-positioned display modes
- Uses local files only

## Media Pack Format

The plugin expects a local frame pack folder containing a `frames` subfolder.

Example:

```text
MyFramePack/
└─ frames/
   ├─ frame_000001.jpg
   ├─ frame_000002.jpg
   ├─ frame_000003.jpg
   └─ ...
```

Supported frame formats:

- `.jpg`
- `.jpeg`
- `.png`

## Notes

This plugin is only a local visual display tool. It does not include media, fetch media, recommend media sources, or provide hosted content. All files must already exist locally on the user's computer.
